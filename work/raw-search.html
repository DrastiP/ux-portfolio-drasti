Raw Search Data Visualization

Designing a powerful, intuitive interface for searching through billions of security logs in real-time.

Role

Lead Designer

Timeline

5 months · 2023

Team

Search & Data · Eng

Org

Arctic Wolf

01 — Context

Searching the Void

Security investigations often require digging into "Raw Search"—the massive repository of every log entry collected across an organization's network. The existing interface was technical and required complex query language knowledge, making it inaccessible to less experienced users.

02 — The problem

Information Overload

When searching through billions of logs, users were often overwhelmed by the volume of data. The challenge was to create an interface that provided both a "bird's-eye view" of trends and the ability to "drill down" into specific, microscopic details without losing orientation.

03 — Design Solution

Visual Querying & Log Aggregation

We introduced a visual query builder that allowed users to drag and drop filters, reducing the reliance on technical syntax. Additionally, we implemented a dynamic histogram that visualized log volume over time, helping users quickly identify spikes in activity that might indicate a breach.

"We turned a command-line experience into a visual investigation tool."
04 — Outcomes

Search at Scale

Billions Logs searchable in seconds
40% Increase in user adoption
Faster Investigation speed
